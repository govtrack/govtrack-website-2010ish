WMS API: upload-geometry

The upload-geometry.cgi API access method is used to create geographic features, using longitude-latitude coordinates, in what is called a "dataset". Datasets lack any visual styling information, which is instead set in "layers". This form facilitates manual calling of this API method, although the API can be called automatedly through an HTTP POST interface.
