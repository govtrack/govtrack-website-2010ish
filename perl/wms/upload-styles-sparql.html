WMS API: upload-styles with SPARQL

The upload-styles-sparql.cgi API access method is used to assign visual presentation styling to geographic features in a particular layer by getting value data with a SPARQL query. This form facilitates manual calling of this API method, although the API can be called automatedly through an HTTP POST interface.
